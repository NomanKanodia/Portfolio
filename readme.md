# Portfolio website using Flask and Jinja

This is my portfolio website developed by using HTML, CSS, Flask and Jinja. It display my personal information, my skills, my education and my projects which can be used directly from the website.

## How to run:-

### Step 1:- Make sure Python is installed in your computer.

### Step 2:- Create Virtual Environment
python3 -m venv venv

source venv/bin/activate

### Step 3:- Install flask
pip install flask

### Step 4:- Run the Flask App
python app.py

### Step 5:- Once the server starts, open the URL shown in terminal.
